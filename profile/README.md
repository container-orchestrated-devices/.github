# Container Orchestrated Device Working Group

The Container Orchestrated Device (COD) Working Group is a working group of TAG Runtime.
The primary focus of the working group is driving the development and adoption of technologies
that simplify the use of complex devices such as GPUs and accelerators in containerized and other
environments.
